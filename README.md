[ENG]

A simple Python script designed for automatic file segregation in a specified folder.
The program identifies file extensions (e.g. pdf, jpg, zip, png, txt, rar) and moves them into organized subfolders or creates the folders if they doesn't exist.

# Tech Stack
- Python
- os library -> for path and file management
- shutil library -> for moving files

# Script Functions
Automated file segregation allows for quick folder organization, improving navigation and ensuring significant time savings when working with large volumes of files.

# How to run:
- Configure the 'folder_to clean' variable by providing the path to the folder you want to organize
- Run the script in your Python environment
- The folder is now organized

[PL]

Prosty skrypt napisany w języku Python, służący do automatycznego segregowania plików we wskazanym folderze.
Program rozpoznaje rozszerzenia plików (np. txt, pdf, jpg, png, zip, rar) i tworzy dla nich podfoldery lub przenosi do odpowiednich, jeśli już istnieją.

# Wykorzystane narzędzia:
- Python (VSCode)
- Biblioteka os -> do obsługi ścieżek i plików
- Biblioteka shutil -> do fizycznego przenoszenia plików

# Funkcje skryptu:
Automatyzacja segregacji plików pozwalająca w szybki sposób uporządkować dany folder ułatwiając nawigację i gwarantując oszczędność czasu przy pracy z większą ilością plików.

# Uruchamianie
- Skonfiguruj zmienną 'folder_to_clean', podając ścieżkę do folderu, który chcesz uporządkować
- Uruchom skrypt w swoim środowisku
- Folder został uporządkowany
